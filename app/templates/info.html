Table Descriptions


This platform allows for the organisation of data required by the NCP funding managment.
This page explains what information is captured in each table.
For more information and instructions please refer to the user manual.

Table Name Description Connections
Students Contains each student's personal information (who have been paid a grant). Universities & Campuses
Programs Contains all the funding available for ACICIS programs. No connections
Universities Contains basic information of past and present ACICIS member Universities. No connections
Campuses Contains basic information of past and present ACICIS member University Campuses. Universities
Payments Contains information of all past payments to students. Students & Programs
Grants Contains information of all grants awarded or tentatively awarded to students and corresponding payment details. Programs, Students, Universities & Campuses
Grants By University Contains summarised information regarding each universities allocated grants. Universities
Programs By University Contains information regarding each universities allocated programs Programs, Universities & Grants By University
